### Week 4 objectives:

* **Consolidate Week 3 knowledge:** IAM roles for EC2, Cloud9, Amazon RDS, and basic CloudWatch.
* **Get familiar with Amazon Lightsail** to quickly deploy lightweight applications and optimize costs.
* **Master automated scaling mechanisms (EC2 Auto Scaling)**, load balancing, advanced monitoring with CloudWatch, and domain routing with Route 53.

### Week 4 achievements:

* Deploy and package applications quickly using Amazon Lightsail, knowing how to set up basic security and backups.

* Proficient in infrastructure architectures with automated scaling (Auto Scaling) and load balancing capabilities to ensure high availability.

* Good usage of advanced monitoring suite in CloudWatch (Dashboard, Logs, Alarms) to proactively control system state.

* Master the mindset of setting up cloud DNS systems and routing traffic using Amazon Route 53.
