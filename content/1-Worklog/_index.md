This page summarizes my **12-week** worklog in the First Cloud Journey (FCAJ) program, following the [Cloud Journey](https://cloudjourney.awsstudygroup.com/) learning path from AWS Study Group. Each week records objectives, tasks completed, timeframe, and outcomes.

The full program was completed over about **3 months** (**April 20–July 12, 2026**), covering AWS fundamentals, workload migration, system optimization, application modernization, containers, and data analytics. *The official AWS Vietnam internship period is May 4–July 12, 2026; the first weeks (April 20–May 3) were onboarding and preparation before the internship began.* Weekly contents:

**Week 1:** [Introduction to cloud concepts, EC2 virtualization, AWS Console, and CLI](1.1-week1/)

**Week 2:** [Cost management with Budgets, IAM security, VPC networking, and S3 storage](1.2-week2/)

**Week 3:** [Secure authorization using IAM Roles, Cloud9 IDE, RDS setup, and CloudWatch monitoring](1.3-week3/)

**Week 4:** [Deploying on Lightsail, setting up Auto Scaling with ALB, advanced monitoring, and Route 53 DNS](1.4-week4/)

**Week 5:** [AWS Support tiers, advanced CLI operations, and building a Highly Available web architecture](1.5-week5/)

**Week 6:** [Migration frameworks, VM Import/Export tools, database migration using DMS, and SCT](1.6-week6/)

**Week 7:** [Performance optimization, Lambda serverless, Systems Manager automation, and CloudFormation IaC](1.7-week7/)

**Week 8:** [Reviewing systems based on Security, Reliability, Performance, and Cost pillars of AWS Well-Architected](1.8-week8/)

**Week 9:** [Modernizing applications using Microservices, Serverless architectures, and CI/CD pipelines](1.9-week9/)

**Week 10:** [Containerizing apps with Docker, publishing to ECR, and orchestrating with ECS, Fargate, and EKS](1.10-week10/)

**Week 11:** [Practicing Dockerfile optimization and deploying containers to ECS and EKS clusters](1.11-week11/)

**Week 12:** [Big Data workflows including S3 Data Lake, Glue Catalog, Athena queries, and QuickSight BI dashboards](1.12-week12/)
