This page summarizes my **12-week** worklog in the First Cloud Journey (FCAJ) program, following the [Cloud Journey](https://cloudjourney.awsstudygroup.com/) learning path from AWS Study Group. Each week records objectives, tasks completed, timeframe, and outcomes.

The full program was completed over about **3 months** (**April 20–July 12, 2026**), covering AWS fundamentals, workload migration, system optimization, application modernization, containers, and data analytics. *The official AWS Vietnam internship period is May 4–July 12, 2026; the first weeks (April 20–May 3) were onboarding and preparation before the internship began.* Weekly contents:

**Week 1:** [Getting familiar with AWS, EC2, AWS Console, and AWS CLI](1.1-week1/)

**Week 2:** [AWS Budgets, IAM, VPC, and Amazon S3](1.2-week2/)

**Week 3:** [IAM Role for EC2, Cloud9, RDS, and CloudWatch](1.3-week3/)

**Week 4:** [Lightsail, Auto Scaling, advanced CloudWatch, and Route 53](1.4-week4/)

**Week 5:** [AWS Support, CLI on EC2, and highly available web application workshop](1.5-week5/)

**Week 6:** [Migration strategy, VM Import/Export, DMS, and SCT](1.6-week6/)

**Week 7:** [System optimization, Lambda, Systems Manager, and CloudFormation](1.7-week7/)

**Week 8:** [Security, reliability, performance, and cost optimization on AWS](1.8-week8/)

**Week 9:** [Application modernization, microservices, serverless, and CI/CD](1.9-week9/)

**Week 10:** [Docker, ECR, and container services (ECS, Fargate, EKS)](1.10-week10/)

**Week 11:** [Hands-on packaging and deploying containerized applications on AWS](1.11-week11/)

**Week 12:** [Data & Analytics: Data Lake, Glue, Athena, Redshift, and QuickSight](1.12-week12/)
