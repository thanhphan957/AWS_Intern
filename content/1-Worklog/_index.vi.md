Trong thời gian **12 tuần thực tập**, tôi đã từng bước làm quen với môi trường làm việc, học tập các dịch vụ của AWS, nghiên cứu kiến trúc Serverless và tham gia xây dựng dự án nhóm Serverless Todo API. Bên cạnh việc hoàn thành các nội dung đào tạo, tôi còn thực hành triển khai ứng dụng trên AWS, phát triển API, thiết kế giao diện, kiểm thử hệ thống và hoàn thiện tài liệu kỹ thuật cho dự án.

Các nội dung chính được thực hiện trong từng tuần như sau:

**Tuần 1:** [Tham gia định hướng, tìm hiểu Cloud Computing, tạo tài khoản AWS Free Tier và EC2 Linux (17/04/2026 - 23/04/2026)](1.1-week1/)

**Tuần 2:** [Tìm hiểu RDBMS, NoSQL, thiết kế cơ sở dữ liệu quan hệ và khởi tạo RDS MySQL (24/04/2026 - 30/04/2026)](1.2-week2/)

**Tuần 3:** [Nghiên cứu DynamoDB, thao tác AWS CLI, Generative AI, LLMs và Amazon Bedrock (01/05/2026 - 07/05/2026)](1.3-week3/)

**Tuần 4:** [Nghiên cứu AWS IAM, xây dựng VPC tùy chỉnh, Bastion Host và NAT Gateway (08/05/2026 - 14/05/2026)](1.4-week4/)

**Tuần 5:** [Quản lý chi phí AWS, Pricing Calculator, AWS Budgets và cảnh báo chi phí (15/05/2026 - 21/05/2026)](1.5-week5/)

**Tuần 6:** [Lựa chọn đề tài AWS Serverless Todo API, nghiên cứu kiến trúc hệ thống và SRS (22/05/2026 - 28/05/2026)](1.6-week6/)

**Tuần 7:** [Thiết kế DynamoDB schema, API Specification, AWS SAM và DynamoDB Local (29/05/2026 - 04/06/2026)](1.7-week7/)

**Tuần 10:** [Xây dựng UI Frontend, Cognito SDK, S3 Static Hosting, CloudFront & ACM (19/06/2026 - 25/06/2026)](1.10-week10/)

**Tuần 11:** [Cấu hình GitHub Actions CI/CD, DynamoDB TTL, tài liệu kỹ thuật & README (26/06/2026 - 02/07/2026)](1.11-week11/)
