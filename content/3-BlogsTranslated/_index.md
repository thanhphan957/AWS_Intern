## Blog Post 2: Cost Optimization Strategies on AWS

**Title**: "Save Money on AWS: 5 Proven Cost Optimization Strategies"

**Published On**: AWS Study Group (June 2026)

**Key Points**:
- Use AWS Free Tier effectively
- Choose the right compute service (EC2 vs Lambda vs Fargate)
- Implement auto-scaling policies
- Clean up unused resources regularly
- Monitor costs with CloudWatch and Cost Explorer

**Strategies Covered**:
1. Reserved Instances for predictable workloads
2. Spot Instances for fault-tolerant applications
3. DynamoDB on-demand vs provisioned capacity
4. S3 Intelligent-Tiering
5. VPC Endpoints to avoid data transfer charges

**Learning Outcomes**:
- Practical cost management techniques
- How to calculate ROI for AWS services
- Best practices for billing alerts

## Blog Post 3: AWS Security Best Practices

**Title**: "Secure Your AWS Apps: TL;DR Guide to IAM and Least Privilege"

**Published On**: AWS Study Group (July 2026)

**Key Points**:
- Principle of Least Privilege explained
- IAM roles vs policies
- How to avoid common security mistakes
- Encryption at rest and in transit
- CloudTrail for audit logging

**Security Topics**:
- Never use root account for daily work
- Enable MFA on all accounts
- Create specific IAM roles for applications
- Rotate credentials regularly
- Use KMS for key management

**Learning Outcomes**:
- Applied security best practices in workshop
- Implemented IAM policies correctly
- Understood compliance requirements
