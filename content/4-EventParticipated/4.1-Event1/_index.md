# Event Summary Report: "AWS Cloud, DevOps & Orientations for Cloud Engineers"

---

### 1. Purpose of the Event

- Explore real-world responsibilities of Data Analytics Engineers and DevOps Engineers in enterprise environments.
- Learn global working standards and corporate culture within Multi-National Corporations (MNCs).
- Understand career growth pathways in the AWS Cloud ecosystem and engage with AWS Community networks.
- Analyze scalable system design principles through a URL Shortening Service case study on AWS.

---

### 2. Speaker List

- **Mr. Dat & Mr. Cuong**: Data Analytics Engineers - Data Analytics & MNC Corporate Culture Experts.
- **Mr. Hoang Trong**: Senior DevOps Engineer - Infrastructure & CI/CD Pipeline Specialist.
- **Mr. Hieu Nghi**: AWS Community Builder & Solutions Architect - First Cloud Journey to AWS Partner.
- **Mr. Kien & Mr. Tho**: Cloud System Architects - Scalable Architecture Specialists.

---

### 3. Highlighted Topics

#### Topic 1: Real-world Stories & Corporate Culture at MNCs (Mr. Dat & Mr. Cuong)
- Root Cause Analysis and strategic business proposals in Data Analytics Engineering.
- Essential professional skills: Critical Thinking, Effective Communication, Data Storytelling, Problem Solving.
- 5-Stage Growth Path: Follower → Learner → Problem Solver → System Thinker → Super Star.
- MNC Culture Pillars: No-Blame culture, Caring & Inclusive workplace, Kaizen continuous improvement.

#### Topic 2: What Does a DevOps Engineer Really Do? (Mr. Hoang Trong)
- DevOps culture bridging Development and Operations across SDLC.
- Core technical stack: Linux, Networking, Git, CI/CD, Containerization, Scripting.
- DevOps mindset: Ask *"Why"* before *"How"*, automate repetitive tasks, leverage AI productivity tools.

#### Topic 3: From First Cloud AI Journey to AWS Partner (Mr. Hieu Nghi)
- Career growth roadmap from student developer to AWS Partner Solutions Architect.
- Community programs: AWS Student Builder & AWS Community Builder.
- Career path requirements for Solutions Architect, DevOps, and Platform Engineers.

#### Topic 4: A Scalable URL Shortening Service on AWS (Mr. Kien & Mr. Tho)
- High-availability, sub-10ms latency system design for URL shortening.
- AWS Tech Stack: API Gateway, Lambda (Base62 encoding), DynamoDB (Key-Value), ElastiCache (Redis), CloudFront CDN.

---

### 4. Key Learnings & Application

- Mastered System Thinking and cloud design principles: Decoupling, Caching, and NoSQL partitioning.
- Applied GitHub Actions CI/CD, DynamoDB TTL, and CloudWatch monitoring to the Serverless Todo API project.
