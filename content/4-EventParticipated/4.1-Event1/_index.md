# Event 1 Summary Report: "Real-world MNC Experience & What a DevOps Engineer Really Does"

---

### 1. Purpose of the Event

- Explore real-world responsibilities of Data Analytics Engineers and DevOps Engineers in enterprise environments.
- Learn global working standards and organizational culture within Multi-National Corporations (MNCs).
- Understand career growth trajectories from executor to system thinker.
- Master Software Development Life Cycle (SDLC) fundamentals, CI/CD automation, and AI productivity tools.

---

### 2. Speaker List

- **Mr. Dat & Mr. Cuong**: Data Analytics Engineers - Data Analytics & MNC Corporate Culture Experts.
- **Mr. Hoang Trong**: Senior DevOps Engineer - Infrastructure & CI/CD Pipeline Specialist.

---

### 3. Highlighted Topics

#### Topic 1: Real-world Stories & Corporate Culture at MNCs
- **Day-to-day Data Analytics Engineering**: Beyond reporting numbers—focusing on Root Cause Analysis and strategic business proposals.
- **Key Professional Skills**: Critical Thinking, Effective Communication, Data Storytelling, and Problem Solving.
- **5-Stage Career Path**: Follower → Learner → Problem Solver → System Thinker → Super Star.
- **MNC Culture Pillars**: No-Blame culture, Caring & Inclusive workplace, and Continuous Improvement (Kaizen).

#### Topic 2: What Does a DevOps Engineer Really Do?
- **Demystifying DevOps**: DevOps is a culture bridging Development and Operations to accelerate release velocity.
- **Core Technical Stack**: Linux Administration, Networking, Git, CI/CD, Containerization (Docker/Kubernetes), and Python/Golang scripting.
- **SDLC Mastery**: Managing Build, Test, Deploy, Logging, Configuration, and Environment Variables.
- **DevOps Mindset**: Ask *"Why"* before *"How"*, automate repetitive tasks, and leverage AI to boost productivity without sacrificing problem-solving skills.

---

### 4. Key Learnings & Takeaways

- **System Thinking**: Gained macro-level perspective on software architecture and infrastructure automation.
- **No-Blame Culture**: Embraced blameless post-mortems focused on root-cause fixes.
- **Data Storytelling**: Learned how to present complex technical data into actionable insights.

- Applied Git Flow and GitHub Actions CI/CD to automate AWS SAM deployments.
- Implemented Environment Variable configurations and Infrastructure as Code best practices.
